⭐ Key Innovation #1

AI Pre-Consultation Copilot

Before the appointment, AI synthesizes symptoms + reports into a structured clinical brief the doctor reviews before the patient walks in.

  • 🗣️
    Accepts symptoms in Hindi, Marathi, English
  • 📊
    Correlates with uploaded report data
  • 🚨
    Assigns urgency: LOW → CRITICAL
  • 🎯
    Suggests focus areas + recommended specialty
  • 🔬
    Flags abnormal lab values with normal ranges
  • 📈
    Analysis reliability indicator (HIGH / MEDIUM / LOW)

⚠️ Advisory output only — not a clinical diagnosis. Always reviewed by the consulting physician.

🔄 AI Resilience

4-tier model fallback: gemini-flash-latest → 2.0-flash-lite → 2.0-flash → 2.5-flash. Daily quota exhaustion is cached — no wasted API calls. Per-minute rate limits use server-suggested retry delays. Safety blocks throw immediately, never retry.

🛡️ Security

Helmet.js security headers, MongoDB injection sanitization, HTTP parameter pollution prevention, CORS origin whitelist, role-based rate limiting (auth: 20/15min, AI: 20/15min, uploads: 10/15min), medical disclaimers on all AI outputs.
